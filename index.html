JavaScript - Desafio Clase 3

Algoritmo

Es un conjunto de procedimientos o funciones que se necesitan para realizar cierta operacion o accion.

Condicionales

Conjunto de instruccion o sentencias que nos permiten determinar si se ejecuta o no parte del programa. Es decir si sucede tal condicion lo hago o no, se ejecuta o no se ejecuta.

Que emplee bucles o instrucciones condicionales

Ciclos, bucles o iteraciones, sirven para hacer alguna operacion en forma repetida, es decir, repetir la operacion mas de una vez en el programa. En JavaScript tenemos los bucles: for que repiten el bloque de codigo una cierta cantidad de veces; ciclos while y do while, que repiten mientras la condicion sea verdadera.
